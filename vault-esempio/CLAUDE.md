# CLAUDE.md - Vault di esempio (esercitazione igiene)

Questo è un **vault di esempio** usa-e-getta, già un po' "vissuto": serve per provare le skill di igiene (`/stato-vault`, `/stato-memory`) senza rischiare niente su un vault vero. Dentro c'è del disordine messo apposta - link che non portano da nessuna parte, una memoria con due decisioni che si contraddicono, una mappa non aggiornata, un file fuori posto.

> **Per Claude.** Quando lo studente lancia una skill di igiene qui dentro, trattalo come un vault vero: trova i problemi, mostrali, e proponi i fix una cosa alla volta con conferma.

## Com'è organizzato

- `Progetti/` - i progetti/clienti (qui ce n'è uno: **Studio Rossi**, uno studio di commercialisti).
- ogni progetto ha: `CLAUDE.md` (chi è), `MEMORY.md` (decisioni/numeri), un file-mappa con lo stesso nome del progetto (il MOC), e cartelle `calls/` e `reports/`.
- `Daily/Journal/` - le note di giornata.
