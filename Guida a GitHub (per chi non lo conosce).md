# Guida a GitHub (per chi non lo conosce)

Se hai ricevuto questa cartella e non sai cos'è GitHub, leggi qui. Se l'hai già scaricata e aperta, puoi saltare: sei a posto.

**GitHub** è solo un posto dove mettiamo a disposizione dei file da scaricare. Non devi creare account né imparare nulla di tecnico.

## Scaricare la cartella (il modo più semplice)

1. Apri il link che ti abbiamo dato (porta a una pagina GitHub).
2. Cerca il pulsante verde **`Code`** (in alto a destra nell'elenco dei file).
3. Clicca **`Download ZIP`**.
4. Scompatta lo ZIP (doppio clic) dove preferisci, per esempio sul Desktop.
5. Dentro trovi la cartella dell'esercitazione, con il file **`INIZIA QUI.md`**. Apri quello per primo.

## E adesso?

Apri **Claude Code** in quella cartella e segui le istruzioni di `INIZIA QUI.md`. Da lì in poi ti guida Claude.

## Se qualcosa non torna

- Non trovi il pulsante `Code`? Assicurati di essere sulla pagina principale del progetto (non dentro un singolo file).
- Lo ZIP non si apre? Riprova il download, a volte si interrompe.
- Nel dubbio, scrivi nel gruppo del corso: ti diamo una mano.
