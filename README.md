# Corso Claude Code - Esercitazione Lezione 4

Benvenuto. Qui trovi tutto per l'esercitazione dell'ultima lezione: tenere in salute e far crescere il tuo secondo cervello, e costruire la tua prima app.

## Come si parte (3 mosse)

1. **Scarica questa cartella**: bottone verde **Code** → **Download ZIP**, poi scompattala. (Se non sai come si fa, apri `Guida a GitHub (per chi non lo conosce).md`.)
2. **Apri Claude Code** dentro la cartella scaricata.
3. **Apri il file `INIZIA QUI.md`** e segui da lì: ti guida Claude, passo per passo.

## Cosa c'è dentro

- **`INIZIA QUI.md`** - il punto di partenza (leggi questo).
- **`Guida esercitazione 1 - Igiene e crescita.md`** e **`Guida esercitazione 2 - La tua prima app.md`** - i due percorsi.
- **`Guida soul - attivare la memoria che ti conosce.md`** - la memoria più profonda.
- **`Da leggere/`** - le due parti della lezione in versione testo.
- **`skill-pronte/`** - le skill da installare.
- **`vault-esempio/`** - un vault già "vissuto" su cui provare l'igiene senza rischi.
- **`mini-crm-partenza.xlsx`** - il foglio da cui parte la tua prima app.

Buon lavoro. Quando sei pronto, apri **`INIZIA QUI.md`**.
